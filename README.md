## 🚀 Projects I'm Proud Of

### 🎂 [Sweet N Sugary](https://github.com/Krishypatel-4061/SweetNSugary)
A full-stack home bakery web app for a real business in Jamnagar.

The interesting part: customers can upload a photo of any cake and **Gemini Vision AI auto-configures a 3D model** of it in real-time using React Three Fiber. Built with Next.js 14, TypeScript, Neon PostgreSQL, JWT auth, and an admin Kanban dashboard with WhatsApp order updates.

`Next.js` `TypeScript` `Gemini AI` `React Three Fiber` `PostgreSQL` `Tailwind`

---

### 💰 [SmartRupee](https://github.com/Krishypatel-4061/SmartRupee-Hackathon)
Hackathon project focused on personal finance for Indian users. Built fast, shipped under pressure.

`HTML` `JavaScript`

---

### 💊 [Pharma Assist](https://github.com/Krishypatel-4061/Pharma-Assist)
A desktop application for pharmacy management — inventory, billing, and patient records — built with Python, Tkinter, and MySQL. Focused on making it genuinely usable for a non-technical pharmacy owner.

`Python` `Tkinter` `MySQL`

---

### 🤖 [Holo Battalion — e-Yantra](https://github.com/Krishypatel-4061/holo-battalion-eyantra)
Robotics competition project from IIT Bombay's e-Yantra initiative. Autonomous navigation and task execution on a holonomic drive robot.

`C++` `Embedded Systems` `Robotics`
